
# PROGRAM 2: BREADTH FIRST SEARCH

###(Name): Yafet Kubrom (EUID):11334602

This program takes integers adjacency list format as input file (data2.txt) 
and output if the graph is connected or not to out2.txt.

## Assumption

* `data2.txt` must exist in directory.
* if n vertices, there must be n lines (beside blank lines).
* the numbers must be integers.
* `out2.txt` is optional. The program will create one if one does not exist.

## Compile & Execute

Part A

* Compile using `g++ -o bfs bfs.cpp`
* Execute using `./bfs`
