
# PROGRAM 3: HEAP SORT

## (Name): Yafet Kubrom (EUID):11334602

Program 3 implements heap sort using C++ in O(N logN)

## Assumption

* `data3.txt` must exist in directory.
* `data3.txt` must have 100 or less numbers per line
* `output3.txt` is optional. The program will create one if one does not exist.

## How to Compile and Execute

* Compile using `g++ -o heapSort heapSort.cpp`
* Execute using `./heapSort`
